wafer.run

a wafer thin runtime

Block-Based Architecture

Build your backend from composable blocks. Each block handles a message and returns a result. Chain them together for complex workflows.

Multi-Language Blocks

Write blocks in Rust, Go, or JavaScript. Compile to WebAssembly and run them all in the same runtime.

Built-in Services

Database, storage, crypto, logging, config, and network services are built-in. Blocks access them through a clean service interface.

Transport Agnostic

The runtime is a pure executor with zero transport knowledge. HTTP, WebSocket, MQTT — each transport is just a bridge in wafer-core or an external adapter.

Shared Block Ecosystem

Use pre-built blocks from wafer-core: CORS, rate limiting, security headers, auth, monitoring, and more.

Single Binary Deploy

Compile everything into a single binary. Deploy to Fly.io, Railway, Docker, or bare metal.
