Quick Start

Create and run your first block in under a minute.

1. Install the CLI

npm install -g wafer-run

2. Create a Project

wafer init my-app
cd my-app
npm install

This scaffolds a project with a hello-world block, chain config, and build setup.

3. Your Block

Open blocks/hello/index.ts — the generated starter block:

import { Block, BlockInfo, Context, Message, Result, register } from "wafer-sdk-ts";

class HelloBlock extends Block {
    info(): BlockInfo {
        return { name: "hello", version: "1.0.0", interface: "handler@v1", summary: "Says hello" };
    }

    handle(ctx: Context, msg: Message): Result {
        return msg.respond(200, '{"hello":"world"}', "application/json");
    }
}

register(new HelloBlock());

4. Run It

wafer dev

This builds your block to WASM and starts a dev server with live reload on port 8080.

5. Test It

curl http://localhost:8080
# {"hello":"world"}
