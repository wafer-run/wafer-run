Clients API

Typed wrapper functions in wafer_run::clients for calling infrastructure blocks. All functions take ctx: &dyn Context as their first argument and return Result<T, WaferError>. WASM blocks access the same services via typed clients in the Guest SDK.

database (wafer_run::clients::database)

FunctionDescription
get(ctx, collection, id) → RecordGet a record by ID
list(ctx, collection, opts) → RecordListList records with filtering and pagination
create(ctx, collection, data) → RecordCreate a new record
update(ctx, collection, id, data) → RecordUpdate an existing record
delete(ctx, collection, id)Delete a record
count(ctx, collection, filters) → i64Count matching records
sum(ctx, collection, field, filters) → f64Sum a numeric field
query_raw(ctx, query, args) → Vec<Record>Execute a raw SQL query
exec_raw(ctx, query, args) → i64Execute a raw SQL statement
get_by_field(ctx, collection, field, value) → RecordGet first record matching a field value
upsert(ctx, collection, field, value, data) → RecordCreate or update by field match
delete_by_field(ctx, collection, field, value)Delete records matching a field value

storage (wafer_run::clients::storage)

FunctionDescription
put(ctx, folder, key, data, content_type)Upload an object
get(ctx, folder, key) → (bytes, ObjectInfo)Download an object
delete(ctx, folder, key)Delete an object
list(ctx, folder, opts) → ObjectListList objects in a folder
create_folder(ctx, name)Create a storage folder
delete_folder(ctx, name)Delete a storage folder
list_folders(ctx) → Vec<FolderInfo>List all folders
