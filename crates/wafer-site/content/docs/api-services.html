Platform Services

Services available through Context in native blocks. WASM blocks access these via typed clients in the Guest SDK.

DatabaseService

MethodDescription
get(collection, id) → RecordGet a record by ID
list(collection, opts) → RecordListList records with filtering and pagination
create(collection, data) → RecordCreate a new record
update(collection, id, data) → RecordUpdate an existing record
delete(collection, id)Delete a record
count(collection, filters) → i64Count matching records
sum(collection, field, filters) → f64Sum a numeric field
query_raw(query, args) → Vec<Record>Execute a raw SQL query
exec_raw(query, args) → i64Execute a raw SQL statement

StorageService

MethodDescription
put(folder, key, data, content_type)Upload an object
get(folder, key) → (bytes, ObjectInfo)Download an object
delete(folder, key)Delete an object
list(folder, opts) → ObjectListList objects in a folder
create_folder(name, public)Create a storage folder
delete_folder(name)Delete a storage folder
list_folders() → Vec<FolderInfo>List all folders

CryptoService

MethodDescription
hash(password) → StringHash a password (bcrypt)
compare_hash(password, hash)Verify a password against a hash
sign(claims, expiry) → StringCreate a signed JWT token
verify(token) → HashMapVerify and decode a JWT token
random_bytes(n) → Vec<u8>Generate random bytes

NetworkService

MethodDescription
do_request(req) → ResponseMake an outbound HTTP request

LoggerService

MethodDescription
debug(msg, fields)Log at debug level
info(msg, fields)Log at info level
warn(msg, fields)Log at warn level
error(msg, fields)Log at error level

ConfigService

MethodDescription
get(key) → Option<String>Read a config value
set(key, value)Write a config value
