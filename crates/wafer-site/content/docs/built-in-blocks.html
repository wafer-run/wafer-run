Built-in Blocks

wafer ships with several ready-to-use blocks in the @wafer namespace:

  • @wafer/security-headers — adds standard security headers (X-Frame-Options, CSP, etc.)
  • @wafer/cors — handles CORS preflight and response headers
  • @wafer/rate-limit — token-bucket rate limiting per client IP
  • @wafer/readonly-guard — rejects write operations (POST, PUT, DELETE) in read-only mode
  • @wafer/monitoring — request logging, timing, and metrics collection
  • @wafer/auth — JWT/Bearer token validation and user extraction
  • @wafer/iam — role-based access control and permission checks
  • @wafer/web — static file serving with caching and content-type detection
