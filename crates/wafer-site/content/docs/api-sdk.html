Guest SDK

The SDK is available for Rust (wafer-sdk), Go (wafer-sdk-go), and TypeScript (wafer-sdk-ts). WASM blocks implement the same Block interface and use typed service clients to access platform services.

Block Trait (SDK)

Same interface as the runtime — implement these three methods:

MethodDescription
info() → BlockInfoReturn block metadata
handle(ctx, msg) → Result_Process a message
lifecycle(ctx, event) → ResultHandle lifecycle events (optional, default no-op)

Export your block with wafer_sdk::export!(MyBlock) (Rust), wafer.Export(&MyBlock{}) (Go), or register(new MyBlock()) (TypeScript).
