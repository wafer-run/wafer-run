Services

Blocks access built-in services through the Context:

Database

CRUD operations with filtering and pagination:

let db = ctx.service("database");

// Create
db.create("users", &json!({"name": "Alice", "role": "admin"}))?;

// Read
let user = db.get("users", "user-id-123")?;

// List with filtering
let opts = ListOptions {
    filter: Some(json!({"role": "admin"})),
    page: 1,
    page_size: 20,
    sort_by: Some("name".into()),
};
let users = db.list("users", &opts)?;

Storage

Binary object storage (files, images, etc.):

let store = ctx.service("storage");
store.put("avatars/alice.png", &bytes)?;
let data = store.get("avatars/alice.png")?;
let files = store.list("avatars/")?;

Other Services

  • Crypto — hashing, token generation, encryption
  • Config — read configuration values
  • Network — make outbound HTTP requests
